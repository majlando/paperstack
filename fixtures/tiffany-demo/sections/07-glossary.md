# Glossary

| Term            | Meaning                                                             |
| --------------- | ------------------------------------------------------------------- |
| Profile         | A WebLager customer; the top of the data hierarchy.                 |
| Box             | A physical archive box, scanned as one unit.                        |
| Document         | A logical document within a box, delimited by a barcode page.       |
| File            | A single scanned page belonging to a document.                      |
| DAO             | Data Access Object — the interface that hides the storage backend.  |
| diamond vision  | The licensed third-party scanning tool Tiffany replaces.            |
| normalside      | Danish academic page unit: 2400 characters; the body length cap.    |
