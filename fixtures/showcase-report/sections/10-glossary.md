# Glossary

| Term | Meaning |
| --- | --- |
| Batch | One production run of a dish; sized by the kitchen, locked at cut-off |
| Cut-off | The time after which orders for today are rejected (10:30 in the pilot) |
| Lock-in | The manager's action freezing batch sizes; after lock-in the tally is a promise |
| Pre-order counter | The separate pick-up point serving only numbered orders |
| Tally | The kitchen's live per-dish count of placed orders |
| Walk-in | A customer buying lunch without a pre-order, through the normal queue |
| Waste report | Daily summary of portions cooked versus picked up, from the audit log |
